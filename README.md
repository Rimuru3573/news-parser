
# 📰 News to Obsidian Parser

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue?logo=python&logoColor=white)](https://www.python.org)
[![Obsidian](https://img.shields.io/badge/Saved%20to-Obsidian-purple?logo=obsidian&logoColor=white)](https://obsidian.md)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Автоматизированный парсер новостей, который собирает свежую информацию, красиво её форматирует и сохраняет прямиком в ваше хранилище (Vault) **Obsidian** в формате Markdown. Идеально для тех, кто ведёт свою базу знаний и хочет оставаться в курсе событий без ручного копипаста.

---

## 🚀 Возможности

* **Автоматический сбор:** Быстрый парсинг актуальных новостей.
* **Интеграция с Obsidian:** Заметки создаются сразу в нужном формате с правильной структурой папок.
* **Чистый Markdown:** Автоматическое конвертирование контента с сохранением заголовков, ссылок и списков.
* **Гибкая настройка:** Всё управление путями и шаблонами вынесено в конфигурационный файл.

---

## 🛠 Инструкция по установке

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/rimuru3573/news-obsidian-parser.git
   cd news-obsidian-parser
    ```

2. **Установите зависимости:**
```bash
pip install -r requirements.txt

```


3. **Настройте окружение:**
Создайте файл `.env` в корневой директории проекта и укажите необходимые переменные.

---

## ⚙️ Настройка (.env)

Для корректной работы парсера в корне проекта обязательно должен лежать файл `.env`. Используйте следующую таблицу для настройки параметров:

| Переменная | Описание | Пример значения |
| --- | --- | --- |
| `OBSIDIAN_PATH` | Абсолютный путь к вашему локальному хранилищу (Vault) Obsidian. | `/home/user/Obsidian/MyVault` или `C:\Users\User\Obsidian\Main` |
| `STANDART_PAGE` | Название дефолтной страницы или папки, куда будут складываться новые заметки. | `News_Inbox` или `Daily_Notes` |

> ⚠️ **Важно:** Убедитесь, что путь в `OBSIDIAN_PATH` указан без кавычек и ведет именно в существующую директорию.

---

## 💻 Использование

Запустите главный скрипт для старта парсинга:

```bash
python main.py

```

После завершения работы скрипта новые файлы появятся в вашем Obsidian в указанной директории. Каждая новость будет оформлена в виде отдельной аккуратной заметки с тегами и ссылкой на источник.
